## 📌 About Meccha Chameleon Vision

**Meccha Chameleon Vision** is a standalone Windows desktop tool for MECCHA CHAMELEON visual experiments.

Designed for testing and educational purposes, this tool allows users to explore visual customization options within the game environment.

## 🎮 Usage

1. Start MECCHA CHAMELEON
2. Launch MecchaChameleonVision.exe
3. Confirm the target process and bridge state in the app
4. Press the saved hotkey

**Settings** are read-only until Edit is selected. Use Save to apply changes or Cancel to discard them.

**Version 1.4.0** uses the mesh-first visual route. Game-derived mesh profiles are prepared locally from the research tools after game updates; they are not ordinary source files.
